## 🎯 完整的测试流程

### 快速开始（3分钟）

1. **环境检查**
   ```powershell
   .\check-environment.ps1
   ```
   确保所有检查都通过

2. **启动所有服务**
   ```powershell
   .\start-all.ps1
   ```
   等待所有服务启动完成

3. **打开前端页面**
   - 浏览器自动打开 http://localhost
   - 点击"连接所有"按钮
   - 查看状态栏确认所有连接（3个绿点）

4. **测试仿真功能**
   - 点击"创建仿真"
   - 点击"订阅"
   - 点击"播放"
   - 观察视频区域的UE画面
   - 观察状态统计的实时更新

5. **使用Navicat查看数据库**
   - 打开Navicat
   - 连接到localhost MySQL
   - 查看metro_sim数据库
   - 查看simulations和simulation_frames表的数据

## 📊 系统架构

```
┌──────────────┐
│  浏览器前端   │ ← 用户界面（player.html）
│  (Chrome)    │
└──────┬───────┘
       │
       ├── HTTP/Socket.IO ──→ ┌──────────────┐
       │                      │  中台服务     │ ← 数据管理和调度
       │                      │  (Express)   │
       │                      └──────┬───────┘
       │                             │ MySQL
       │                             ↓
       │                      ┌──────────────┐
       │                      │  数据库       │ ← 仿真数据存储
       │                      │  (MySQL)     │
       │                      └──────────────┘
       │
       └── WebRTC/PixelStream ──→ ┌──────────────┐
              ↑                    │  UE引擎      │ ← 3D渲染
              │                    │  (UE4.26)   │
              │                    └──────────────┘
              │
        ┌─────┴──────┐
        │ 信令服务器  │ ← WebRTC协商
        │  (cirrus)  │
        └────────────┘
```
